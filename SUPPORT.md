# Support

## Documentation

- **Python API docs**: https://vlrdevapi.pages.dev/docs
- **Official website**: https://vlrdevapi.pages.dev/

## Getting Help

- **Bug reports**: Open an issue at https://github.com/Vanshbordia/vlrdevapi/issues with the `bug` label.
- **Feature requests**: Open an issue with the `enhancement` label.
- **Security issues**: Email vanshb@riftwatch.org — do not file a public issue.

## Community

- **Discussions**: Use GitHub Discussions for questions, ideas, and general help.
- **Contributing**: See [CONTRIBUTING.md](CONTRIBUTING.md) for development setup and PR guidelines.
