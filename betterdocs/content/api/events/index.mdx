## Quick Example

```python title="List ongoing VCT events"
import vlrdevapi as vlr
from vlrdevapi.events import EventTier, EventStatus

# Get ongoing VCT events
events = vlr.events.list_events(
    tier=EventTier.VCT,
    status=EventStatus.ONGOING,
    limit=5
)

for e in events:
    print(f"{e.name} ({e.tier})")
    print(f"  Status: {e.status}")
    print(f"  Prize: {e.prize or 'N/A'}")
    print(f"  Dates: {e.start_text} - {e.end_text}")
```

## Notes

- Many functions return `None` or `[]` on network failures, keeping usage simple.
- Use `EventTier` and `EventStatus` enums for type-safe filtering.
- All date fields are parsed when available, with fallback text representations.
