## Usage

```python title="Access constants (read-only)"
from vlrdevapi import constants

print(constants.VLR_BASE)
print(constants.DEFAULT_TIMEOUT)
```

```python title="Change rate limiting (use helpers)"
import vlrdevapi as vlr

# Set to 5 RPS and enable
vlr.configure_rate_limit(requests_per_second=5.0, enabled=True)

# Restore defaults (10 RPS, enabled)
vlr.reset_rate_limit()
```
