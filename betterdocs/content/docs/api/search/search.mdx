## Signature

```python
import vlrdevapi as vlr

result = vlr.search.search(
    query: str,
    search_type: Literal["all", "players", "teams", "events", "series"] = "all",
    timeout: float = 5.0
) -> SearchResults
```

## Examples

### Unified Search

```python title="Unified search"
import vlrdevapi as vlr

# Search across all categories
results = vlr.search.search("sentinels")

print(f"Total results: {results.total_results}")

# Players
if results.players:
    print("\nPlayers:")
    for p in results.players:
        print(f"  {p.ign} ({p.real_name}) - {p.country}")

# Teams
if results.teams:
    print("\nTeams:")
    for t in results.teams:
        status = "Inactive" if t.is_inactive else "Active"
        print(f"  {t.name} - {t.country} ({status})")

# Events
if results.events:
    print("\nEvents:")
    for e in results.events:
        print(f"  {e.name} - {e.date_range}")
```

### Search Specific Type

```python title="Search specific type"
import vlrdevapi as vlr

# Search only for players
results = vlr.search.search("tenz", search_type="players")

print(f"Found {len(results.players)} players")
for p in results.players:
    print(f"{p.ign}: {p.url}")
```

### Search and Get Details

```python title="Search and get details"
import vlrdevapi as vlr

# Search for teams
results = vlr.search.search("nrg", search_type="teams")

for team in results.teams[:3]:  # First 3 teams
    print(f"Team: {team.name} (ID: {team.team_id}) - {team.country}")
```

## Error Handling

- **Network failures**: Raises `NetworkError`
- **Empty query**: Returns empty SearchResults with `total_results=0`
- **No results found**: Returns SearchResults with empty lists

Always check list lengths before iterating.
